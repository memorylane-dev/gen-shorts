#
# 크롭범위:
#   생략 또는 0  → 9:16 중앙 크롭

#   -500        → 중앙에서 왼쪽으로 500px
#   2~8         → 기준선 2~8 사이 크롭
#
# 제목/브랜드/로고, split/stitch용 secondary_input/secondary_ss 같은 포맷 정보는
# clips.txt가 아니라 formats.json에서 설정
#
# 예시:
# 00:03:26,00:00:39,1_첫번째클립.mp4
